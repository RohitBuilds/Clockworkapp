Clockwork Todo App
Clockwork Todo is a modern full-stack task management application designed to help users organize and manage their daily tasks efficiently. 
The application provides a clean and responsive interface where users can create, track, update, and delete tasks.
This project demonstrates full-stack development by integrating a modern frontend with a powerful backend API and a database.

->Features
Create new tasks with title and description
Mark tasks as completed
Toggle task status (Done / Undo)
Delete individual tasks
Clear all tasks at once
User authentication (Signup & Login)
User-specific tasks (each user sees their own todos)
Responsive and clean UI
